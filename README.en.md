1. **Clone Repo**Ôºö

   ```bash
   git clone https://github.com/ZhuJHua/moodiary.git
   cd moodiary
   ```

2. **Installation dependency**Ôºö

   ```bash

   ```

3. **Running application**Ôºö


4. **Package release**Ôºö
